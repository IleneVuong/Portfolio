Project Layout

Project Layout was assigned to me as a representation of real-world front end development. I was given a prototype designed by a graphic designer and was tasked to code the website, using HTML and CSS, identical to the prototype. This project was also designed for me to practice my skills with flexbox and CSS positioning. It challenged my abilities in attention to details and investigation as I had to replicate every single aspect of the prototype I was given. The most challenging part of the project was positioning a form on top of an image as I was unfamiliar with the concept; however, with persistence, I managed to complete the task.
